# 📖 Información de la Tienda

En la <a href="https://tienda.alohomora.es/" target="_blank">tienda</a> de Alohomora podrás obtener objetos mágicos, rangos y complementos, entre otros artículos.

Todo lo que encuentres en ella es puramente estético dentro de nuestro servidor. ¡Esperamos que te guste!
