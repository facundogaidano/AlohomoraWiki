## Getting Started

Empecemos con lo básico **de Docusaurus**.

## Que vas a necesitar

- Tener **VSCode** instalado.
- Tener **Git y Node.js** instalado.
- Tener una **cuenta en GitHub**.
- Tener **el repositorio de Docusaurus clonado** en tu máquina.

Los detalles de cómo instalar y configurar Git está en esta [Página](./How-To/GitHub.mdx)

## Ubicación y estructura de los archivos

El directorio que más nos va a interesar es el de `docs`. Ahí es donde vamos a tener todos los archivos `.mdx` que vamos a usar para escribir la documentación.
```
AlohomoraWiki
├── docs
│   └── Bienvenida.mdx
│   └── Normas.mdx
│   └── ...
│   └── Docs
│       └── Intro.mdx
│       └── Normas.mdx
│       └── How-To
│           └── CreateDocs.mdx
├── src
│   └── pages
├── docusaurus.config.js
├── ...
```
