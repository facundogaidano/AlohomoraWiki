Para formatear el texto, se pueden usar los siguientes elementos:

## Encabezados

```md
# Encabezado 1

## Encabezado 2

### Encabezado 3

#### Encabezado 4
```

El encabezado nunca lo vamos a usar ya que tenemos [otra alternativa](./Titulo.mdx) para eso.

## Encabezado 2

### Encabezado 3

#### Encabezado 4

## Texto en negrita

Se hace con dos asteriscos `**` al principio y al final del texto.

```md
**Texto en negrita**
```
**Texto en negrita**

## Texto en cursiva

Se hace con un asterisco `*` al principio y al final del texto.

```md
*Texto en cursiva*
```
*Texto en cursiva*

## Texto tachado

Se hace con dos virgulillas `~~` al principio y al final del texto.

```md
~~Texto tachado~~
```
~~Texto tachado~~

## Código

Se hace con tres acentos graves \``` al principio y al final del texto.

```JSX
<code>console.log('Hola mundo')</code>
```
